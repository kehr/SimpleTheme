我在 [Github][0] 上托管了几个小项目，欢迎 star 和 fork，如果你对某个作品有更好的改进想法，请邮件和我联系。

- **[HautClient][3]**  一个校园网802.1x认证客户端的 Linux GUI 版本。使用 C 和 Python 开发。
- **[kehr-vim][4]** 我的 vim 配置。好久没折腾了。
